Per-field override with reset control

Each field shows the effective value (global or overridden). An override indicator + reset button appears when the value differs from global.

Behavior

  • Inherited field: shows global value in muted style, "global" badge. Typing a value creates an override.
  • Overridden field: green left border, "overridden" badge, × reset button. Tooltip on reset shows what the global value is.
  • Reset: clears the repo-level value, field reverts to showing global. One click.
  • Empty override: typing then clearing a field = reset to global (same as clicking reset).
  • Global hint: overridden fields show "Global: ..." below the input for reference.
